# 🍅 Tomato Plant Disease Detection (AI + UX Case Study)

This project uses a **Convolutional Neural Network (CNN)** to identify diseases in tomato plants from images. It represents my work in **Deep Learning** and my focus on **Human-Centered AI**.

---

## 🧠 The AI Logic (The "How")
- **Model:** Convolutional Neural Network (CNN)
- **Framework:** TensorFlow / Keras
- **Environment:** Google Colab
- **Goal:** To provide 90%+ accuracy in detecting Early Blight, Late Blight, and healthy leaves.

## 🎨 The UX Vision (The "User" Side)
As a UI/UX Designer, I believe AI should be accessible. If I were to turn this code into a mobile app, my design focus would be:
- **Instant Feedback:** A simple "Scan" button for farmers in the field.
- **Visual Confidence:** Using color-coded results (Red for disease, Green for healthy).
- **Actionable Steps:** Not just a diagnosis, but clear instructions on how to treat the plant.

## 🛠️ Tech Stack
- **Languages:** Python
- **Libraries:** NumPy, Pandas, Matplotlib, TensorFlow
- **Design Tools:** Figma (for future UI mockups)
